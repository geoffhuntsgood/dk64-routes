# DK64 Speedrun Routes

A simple app that allows for selection of speedrun routes that I use or am interested in using. Hosted at https://geoffhuntsgood.com/dk64-routes.

Each route links to its respective documentation (or, in some cases, the route video).

Once in a route, it can be navigated by scrolling or by jumping to headers by pressing the left and right arrow keys.

The current included routes are Donkey Kong 64 main categories and category extensions, for both N64 and Wii U/NSO. There are also a few extra niche routes as well.
